HTML5 - Principais Elementos

HTML - Definição e História Resumidas:

O HTML5 (Hypertext Markup Language 5) é uma linguagem de marcação de hipertexto utilizada por toda a internet,
e é a estrutura principal de um site, onde se encontram os elementos estáticos como links, imagens e texto que
compõem o que pode ser considerado o esqueleto do contetúdo do site. O HTML original foi lançado em 1991,
por Tim Berners-Lee, que originalmente era usado apenas como uma forma de resolver um problema de comunicação
de pesquisas entre ele e o grupo de colegas do fisíco. Nessa mesma epóca, a internet estava começando a emergir,
e este padrão de marcação de texto ganhou atenção e adoção mundial. Resumindo a história, as especificações em
HTML atualmente vem sendo mantidas pelo World Wide Web Consortium (W3C), que vêm trabalhando com
atualizações da linguagem de marcação de texto desde 1996, com a mais nova atualização sendo
o HTML 5.2, que foi lançado em 2017.

Elementos em HTML:

O documento HTML é composto por elementos, que também podem ser chamados de tags. Estes elementos definem
qual é o tipo de conteúdo ou declaração que será utilizado, e no código eles sempre são declarados com os simbolos
"<" e ">" respectivamente. Um exemplo muito básico é o própio elemento <html> que compõe todo o arquivo HTML.
Os elementos que possuem conteúdo também necessitam de uma tag de fechamento "/" para definir aonde o elemento
acaba. Como exemplo podemos utilizar o elemento <html> citado anteriormente e o elemento <p> que serve
para declarar paragrafos de texto.

Exemplo da tag de fechamento:

<html>
 <p>
 Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore
 magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo
 consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur.
 Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.
 </p>
 </html>

Acima podemos definir que o documento HTML acaba depois do elemento de fechamento </html>, e o elemento
<p> e </p> acaba depois do paragráfo de texto dentro do mesmo. Alguns outros elementos não necessitam de
tag de fechamento e são auto-fechados, principalmente elementos que não abrigam nenhum tipo de conteúdo,
como a tag <meta> que será explicada posteriormente.

<!DOCTYPE html>

É uma declaração obrigatória para definir o arquivo como um documento html. Sempre se encontra no topo
do documento e é definido apenas pelo código <!DOCTYPE html>

A declaração foi simplificada depois da versão 5 do HTML, que permite que utilizamos apenas a palavra chave
html para o documento ser definido como HTML5.

Exemplo:

<!DOCTYPE html>
 <html>
 <p>Conteúdo do arquivo HTML</p>
 </html>
